# 변경 이력

이 프로젝트의 모든 주목할 만한 변경 사항이 이 파일에 문서화됩니다.

형식은 [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)를 기반으로 하며,
이 프로젝트는 [Semantic Versioning](https://semver.org/spec/v2.0.0.html)을 준수합니다.

## [0.1.0] - 2026-02-03

### 추가됨
- 최초 마켓플레이스 릴리스
- INCEPTION, CONSTRUCTION, OPERATIONS의 세 단계로 구성된 핵심 AI-DLC 방법론
- 프로젝트 복잡도에 기반한 적응형 워크플로우 계획
- 채팅 기반 Q&A 상호작용 모델
- 주요 단계 전환 시 승인 게이트
- `aidlc-docs/aidlc-state.md` 를 통한 진행 상황 추적
- 타임스탬프가 포함된 완전한 감사 추적
- 세션 연속성 (재개 기능)
- 역공학을 통한 Brownfield 프로젝트 지원
- 조건부 단계가 있는 단위별 Construction 루프
- Claude Code 플러그인 시스템에 맞게 AWS AI-DLC Workflows에서 조정됨

### 참고 사항
- 이것은 1.0 이전 릴리스입니다 - 파괴적 변경이 발생할 수 있습니다
- AWS AI-DLC Workflows 참조 구현을 기반으로 합니다
